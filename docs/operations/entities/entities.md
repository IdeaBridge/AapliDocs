## Entities

To access the Entities table, click on the "Entities" tab on the top row of the Operations tab group, on the admin home page.

{% include image.html file="entwidget.png" class="docimage" url="https://admin-dev.aapli.app/" alt="Entities Widget" caption="Entities Widget" %}

This table displays all the data tables used in created entities by the user on their suborganisation.
