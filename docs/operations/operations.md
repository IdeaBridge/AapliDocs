## Operations

The Operations tab on the home page provides a comprehensive set of tools (Widgets) and functionalities to streamline and manage operational aspects within Aapli.

{% include image.html file="operations.png" class="docimage" url="https://admin-dev.aapli.app/" alt="Operations" caption="Operations" %}

1. **Users:** This widget provides information and management options related to user accounts. Admin can add and Manage users. 
2. **Applications:** Here, users can view and manage various applications within the Aapli platform.
3. **Entity Builder:** This section allows users to create their own relational databases to display and edit on their website.
4. **Entities:** This produces a list of entities already created by the user, to modify as they wish.
5. **Application Categories:** Users can organize applications into different categories for better organization and classification.
6. **Evaluation Matrix:** This section provides an overview of evaluation criteria and matrices used within Aapli.
7. **Notifications:** Admins can create and customize important notifications and updates through this widget.
8. **Cards:** This allows the user to create convincing verbal strategies for their service.
9. **Announcements:** In this section admin can create and manage important announcements which is to be displayed at users account.
10. **Import/Export:** Users have the option to import or export data or configurations using this widget.
11. **Failed Submissions:** This shows the user a list of submissions from their website that didn't complete their process.
12. **E-mail Logs:** A list of e-mails sent by the website to various users within the suborganisation.
