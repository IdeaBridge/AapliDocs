## E-mail and Notifications

This section enables the Simple Mail Transfer Protocol (SMTP), for admins to send automated e-mails and notifications to users regarding updates on the platform.

To enable SMTP in its entirety, the following details will need to be entered:

(insert image of SMTP panel)

1. **From E-mail Address:** The e-mail address that needs to be used for sending automated e-mails to users.

2. **Display Name:** The name that should appear on these automated e-mails.

3. **Outgoing Mail (SMTP) Server:** The server from which these automated e-mails should be sent.

4. **Port Number:** The port on which the SMTP server should be registered to send these automated e-mails.
